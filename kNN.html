K Nearest Neighbors

K nearest Neighbors is a relatively straight forward machine learning algorithm. It can be used as a classifier or regressor. It holds that the value of a data point is the average of it's K nearest neighbors.

This is a supervised learning technique. The model can be trained on labeled data before making predictions on unseen data points.

Decisions

Determining the Value of K

One consideration that must be made is determining the value of K. Theoretically, K could be set as low as 1, or as high as n. If we think about that for a moment it doesn't really work. K of 1 would not involve any of its neighbors, and in fact doesn't provide any predictive value at all. On the other extreme, if we chose n, then all the data points will be classified as the majority. The best approach would be just select a starting point, and optimize through cross validation.

Addressing Dimensionality

KNN does suffer the curse of Dimensionality. As the number of dimensions increases the nearest nearest neighbor becomes further away. Often times, dimensionality reduction needs to be applied before kNN can be applied.
